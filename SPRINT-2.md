## Major Feature Implemented

### âœ… Import / Export CSV Feature

During Sprint 2, a major new feature was implemented that allows users to **import and export expense data as CSV files**.

### Feature Description:
The Import file + Export CSV functionality allows users to:
- Upload transaction data from a .txt or CSV file into the system.
- Export all stored expenses into a downloadable CSV file.
- Use spreadsheet programs (Excel, Google Sheets) to edit and manage financial information.
- Quickly back up or restore their data.

### Why This Feature Matters:
This feature improves the application by:
- Supporting data portability.
- Allowing users to easily migrate financial records.
- Making data analysis easier.
- Providing backup and recovery options for users.

This enhancement increases the practical value of the Expense Tracker and prepares the system for real-world usage.
